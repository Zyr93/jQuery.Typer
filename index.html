jQuery "Typer" Plugin

This is the demonstration page for the "Typer" plugin for jQuery which, in a nutshell, creates a fake text input element and allows you to fully customize and style both the "editor" itself and parts of the text! For more information, check out the GitHub Repo.

Everything on this page should be a Typer now. If something doesn't work for you, please feel free to inform me at the repo. I'd be glad to fix the first bugs. ;)
